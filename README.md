# Rusty Patched Bumble Bee Decision Tool

An open-source screening tool to support Endangered Species Act Section 7 screening
for the rusty patched bumble bee (Bombus affinis).

## What this tool does
- Collects user inputs through an interactive questionnaire
- Produces a screening determination, risk score (0â€“100), and confidence rating
- Provides an agency-formal explanation, risk drivers, and risk-reduction recommendations

## Disclaimer
This tool is a screening aid and does not replace coordination with the U.S. Fish and Wildlife Service.
Users should retain supporting documentation in the administrative record.
